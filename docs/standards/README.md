# Standards Collection

This directory contains references to standards relevant to the CategoricalReasoner repository and its projects.

## Standards

### W3C Standards

- [RDFC 1.0](https://www.w3.org/TR/rdf-canon/) - RDF Dataset Canonicalization

### IETF RFCs

- [RFC 2119](https://tools.ietf.org/html/rfc2119) - Key words for use in RFCs to Indicate Requirement Levels

### CCSDS

- CCSDS 652.0-M-1 - Mission Operations Services Concept

### ISO Standards

- ISO 14721:2025 - OAIS (Open Archival Information System)
- ISO 16363:2025 - Audit and Certification of Trustworthy Digital Repositories

## Citation Format

All standards should be cited using the format:

```
[Standard Name], [Standard Number], [Version], [Publisher], [Year].
```

Example:
```
RDF 1.1 Concepts and Abstract Syntax, W3C Recommendation, W3C, 2014.
```
