# AGENTS.md - Standards Collection

## Scope
This directory collects relevant standards referenced in the repository's development:
- W3C Standards (RDF, SPARQL, OWL, HTML5)
- ISO Standards (OAIS, COLORE, DOL)
- IETF RFCs (RFC 2119, RFC 3986, RFC 3987)
- Other relevant technical standards

## Standards Collected

### W3C Standards
- RDF 1.1: https://www.w3.org/TR/rdf11-concepts/
- SPARQL 1.1: https://www.w3.org/TR/sparql11-overview/
- OWL 2: https://www.w3.org/TR/owl2-overview/
- RDF Dataset Canonicalization (RDFC 1.0): https://www.w3.org/TR/rdf-canon/

### ISO Standards (Referenced)
- ISO 14721:2025 (OAIS - Open Archival Information System)
- ISO 16363:2025 (Certification of Digital Repositories)
- ISO/IEC 24707 (Common Logic)
- ISO/IEC 17347 (DOL - Description Logic)

### IETF RFCs
- RFC 2119 (Key words for use in RFCs): https://datatracker.ietf.org/doc/html/rfc2119
- RFC 3986 (URI Generic Syntax): https://datatracker.ietf.org/doc/html/rfc3986
- RFC 3987 (IRIs): https://datatracker.ietf.org/doc/html/rfc3987
