# Integration {#sec-integration}

## Connections to Proof Assistants {#subsec-proof-assistants}

Catty integrates with proof assistants including Isabelle/HOL:

- Export of logic definitions
- Import of verified theorems
- Bridge between categorical and type-theoretic presentations

## Semantic Web Integration {#subsec-semantic-web}

The ontology layer enables integration with:

- Wikidata for reference information
- DBpedia for conceptual mappings
- External SPARQL endpoints for query federation

## Interoperability Standards {#subsec-interoperability}

Standard formats ensure interoperability:

| Format | Use | Standard |
|--------|-----|----------|
| RDF/TTL | Ontology exchange | W3C RDF 1.1 |
| OWL | Logic axiomatization | OWL 2 |
| JSON-LD | Web API responses | JSON-LD 1.1 |
| SPARQL | Query language | SPARQL 1.1 |
