# Theorems and Proofs

## Scope
This directory contains standalone theorem and proof documents.

## File Naming Convention
- `thm-{id}.tex` - Theorem statement
- `proof-{id}.tex` - Proof document
- `thm-proof-{id}.tex` - Combined theorem with proof

## Compilation
Each theorem document is self-contained with minimal preamble for standalone compilation.
Use `make theorem` to compile individual theorems.

## Validation
All theorems must pass automated validation.
