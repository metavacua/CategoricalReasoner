# Security Policy

## Supported Versions

Security updates apply to the default branch.

## Reporting a Vulnerability

Please report security issues via GitHub Security Advisories:

1. Go to https://github.com/metavacua/CategoricalReasoner/security/advisories
2. Click **Report a vulnerability** and provide a detailed description.

If you cannot use GitHub Security Advisories, open a private issue with minimal details and maintainers will follow up.
